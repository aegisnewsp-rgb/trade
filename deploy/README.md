## Risk Guardrails (All Scripts)

- **Position size:** ₹7,000 per trade
- **Stop loss:** 0.8% from entry
- **Target:** 4.0× current ATR (Average True Range)
- **Daily loss cap:** 0.3% — skips trading if cap is hit

## How to Run

### Prerequisites

```bash
pip install yfinance requests
```

### Run all scripts

```bash
cd /home/node/workspace/trade-project/deploy
for script in live_*.py; do
  echo "Running $script..."
  python3 "$script"
  echo "---"
done
```

### Run on schedule (cron — market hours only)

```bash
# Run at 9:20 AM IST every weekday
20 9 * * 1-5 cd /home/node/workspace/trade-project && /usr/bin/python3 deploy/live_TCS.py >> deploy/logs/cron_TCS.log 2>&1
```

## Groww API Integration

### Environment variables

```bash
export GROWW_API_KEY="your_api_key_here"
export GROWW_API_SECRET="your_api_secret_here"
```

### How it works

- **Credentials present** → LIMIT order via Groww API (3 retries, exponential backoff)
- **No credentials** → **paper mode**: signal computed, printed, logged to `logs/signals_<SYM>.json`

## Strategy Details

### VWAP (default)
- 14-20 day rolling typical price × volume
- **BUY** when price > VWAP + ATR × 1.5–2.0
- **SELL** when price < VWAP − ATR × 1.5–2.0

### VWAP_RSI_VOL (enhanced)
- VWAP + RSI confirmation + Volume filter
- Higher ATR multiplier (2.0) for fewer but more confident signals

### ADX_TREND
- Average Directional Index with EMA smoothing
- **BUY** when +DI crosses above −DI with ADX > 25

### TSI (True Strength Index)
- Double-smoothed momentum
- **BUY** when TSI crosses above signal line

### Other strategies
- MACD_MOMENTUM, MOMENTUM_DIVERGENCE, FIBONACCI, PARABOLIC_SAR, MA_ENVELOPE
